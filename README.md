# 📈 Trade Journal

Personal trading journal for NSE/BSE Indian markets. Tracks daily P&L, ROI, win rate, consistency score, and P&L distribution chart.

## Features
- Monthly calendar view (green/red per day)
- Daily, weekly, monthly ROI tracking
- Consistency score with arc gauge
- P&L distribution bar chart with running capital curve
- CSV export
- Data stored privately on JSONBin.io

## Local development

```bash
npm install
cp .env.example .env
# Fill in your JSONBin credentials in .env
npm run dev
```
